1. Auto_balloon_&_Auto_BOM_all_sheets.swp
This macro automatically generates balloons and a Bill of Materials (BOM) for all sheets in the drawing, based on the predefined settings.

2. Auto_Block.swp
This macro automatically inserts blocks into the current sheet by using the block file path defined within the drawing.

3. 3sheet_Different_Bom.swp
This macro generates three different types of BOMs — Top-Level Only, Parts Only, and Indented using configurations — each on a separate sheet, using a specified BOM template path. Remaining sheets display the Indented BOM using configurations.

4. Auto_Full_BOM(Page 3Local).swp / Auto_Full_BOM.swp
These macros automatically generate a complete (full) BOM on Page 3 (or the specified sheet), using a local or global BOM template. Suitable for detailed BOM placement based on a standard layout.

5. Sheet1_Auto_balloon.swp
This macro automatically applies balloons to all components on Sheet 1 of the drawing, following the predefined ballooning rules.

6. Toplevel_Bom.swp
This macro creates a Top-Level Only BOM on the current sheet, using a designated template path for formatting.

7. Auto_View.swp
This macro inserts standard drawing views (such as front, top, right, isometric) into the current sheet, based on the model reference and configured layout.

8. Auto_View_block.swp
This macro inserts both standard drawing views and predefined blocks or title blocks into the current sheet, using a specified template or block path.
